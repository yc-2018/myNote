    所以，你可以将路径改为：
    %USERPROFILE%\AppData\Local\Packages\CanonicalGroupLimited.Ubuntu20.04LTS_79rhkp1fndgsc\LocalState
    这样的路径在任何 Windows 用户账户下都可以工作，因为 %USERPROFILE% 会自动被替换为当前用户的用户目录。

int main()
{
int g,s,b;
printf("水仙花数分别有：");
for(int i=100;i<=999;i++){
g = i/1%10;
s = i/10%10;
b = i/100;
if(g*g*g+s*s*s+b*b*b==i)printf("%d\t",i);
}
return 0;
}
