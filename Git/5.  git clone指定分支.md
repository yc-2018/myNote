## ==5.== git clone指定分支

原文：[git clone指定分支](https://golang.0voice.com/?id=5475)

> 在 Git 中，`git clone` 命令可以用来从远程仓库克隆代码到本地。如果想要指定克隆某个特定分支的代码，可以使用以下命令：
>
> ```sh
> git clone -b <branch_name> <remote_repo_url>
> ```
>
> 其中 `-b` 参数用来指定需要克隆的分支名称，`<branch_name>` 是你需要克隆的分支名称， `<remote_repo_url>` 是你需要克隆的远程仓库 URL。
>
> 例如，如果想要从名为 `origin` 的远程仓库中克隆 `dev` 分支的代码到本地，则可以使用如下命令：
>
> ```sh
> git clone -b dev https://github.com/user/repo.git
> ```
>
> 这样就会将 `origin` 远程仓库上的 `dev` 分支对应的代码克隆到本地，并在本地创建一个名为 `repo/` 的文件夹（假设该远程仓库地址为 https://github.com/user/repo.git）。
