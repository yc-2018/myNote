## ==5.==改变提示符颜色

> **注意每个用户都是不一样的文件，所以每个用户可以有不一样的颜色**
>
> 如果你想将提示符(`ikun@yc2023`)变为绿色，你可以在你的bash配置文件（通常是 `~/.bashrc` 或 `~/.bash_profile`）中添加或修改 `PS1` 变量的值。这里是一个例子，它将提示符设置为绿色：
>
> ```bash
> PS1='\[\033[01;32m\]\u@\h\[\033[00m\]:\w\$ '
> ```
>
> 在这个例子中，`\033[01;32m` 是开始设置为绿色的转义序列，`\033[00m` 是重置颜色的转义序列。
>
> 同样，要使改变生效，你需要重新加载bash配置文件。你可以关闭并重新打开终端，或者在终端中运行以下命令：
>
> ```bash
> source ~/.bashrc
> ```
>
> 或者(Ubuntu 是上面其他暂时不知道)
>
> ```bash
> source ~/.bash_profile
> ```
>
> 记住，不同的系统可能使用不同的bash配置文件，你可能需要调整上述命令以适应你的系统。如果你使用的是其他类型的Shell（例如zsh），你可能需要修改其他文件（例如 `~/.zshrc`）。
>
> ---
>
> 在终端中，你可以使用ANSI转义码来设置文本颜色。下面是一些常用的颜色和相应的ANSI转义码：
>
> - 黑色：`0;30`
>
> - 红色：`0;31`
>
> - 绿色：`0;32`
>
> - 黄色：`0;33`
>
> ```bash
>   PS1='\[\033[0;33m\]\u@\h\[\033[00m\]:\w\$ '
> ```
>
> - 蓝色：`0;34`
>
> - 紫色：`0;35`
>
> - 青色：`0;36`
>
> - 白色：`0;37`
>
> 如果你想使用更亮的颜色，可以将前缀从 `0` 改为 `1`。例如，亮红色的代码是 `1;31`。
>
> 这是一个示例，将提示符设置为亮紫色：
>
> ```bash
> PS1='\[\033[01;35m\]\u@\h\[\033[00m\]:\w\$ '
> ```
>
> ---
>
> >==**Linux怎么把[root@yc2023 ~]#的样式更改为绿色**==
> >
> >你可以通过编辑bash提示符 (PS1) 来改变命令行提示符的颜色。这需要在你的 `.bashrc` 文件中添加一些代码。下面是一个例子，它将把提示符更改为绿色：
> >
> >打开 `.bashrc` 文件，你可以使用 `vi` 命令或你喜欢的任何其他编辑器：
> >
> >```bash
> >vi ~/.bashrc
> >```
> >
> >==然后在文件的底部添加以下代码：==
> >
> >```bash
> >export PS1='\[\e[0;32m\][\u@\h \W]\$\[\e[0m\] '
> >```
> >这里是 `PS1` 变量的一些基本元素：
> >
> >- `\u`：当前用户名
> >- `\h`：主机名，==云服务器的就是一坨字符串的，感觉没什么用可以去掉@\h==
> >- `\W`：当前工作目录的基本名称,==如果是小写就是全路径==
> >- `\[\e[0;32m\]`：设置颜色为绿色
> >- `\[\e[0m\]`：重置颜色为默认值
> >
> >>如果是`centOS7`或其他的#（root）和$（其他用户）不会变的话，就用这个（去掉了主机名）
> >>
> >>```bash
> >>export PS1="\[\e[0;32m\][\u \w]\$(if [ \$(id -u) -eq 0 ]; then echo '#'; else echo '$'; fi)\[\e[0m\] "
> >>```
> >
> >保存并关闭文件后，你需要重新加载 `.bashrc` 文件让更改生效：
> >
> >```bash
> >source ~/.bashrc
> >```
> >
> >这样，你的提示符就会变成绿色了。
> >
> >请注意，这些更改只适用于当前用户。如果你需要对所有用户进行此更改，你应该编辑 `/etc/bash.bashrc` 或 `/etc/profile` 文件（取决于你的系统）。
> >
