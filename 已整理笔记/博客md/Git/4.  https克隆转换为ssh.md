## ==4.== https克隆转换为ssh

本来是https 克隆下来的仓库  后来我绑定了ssh 怎么把这个仓库也改为ssh

> 要将已克隆的仓库从 HTTPS 更改为 SSH，可以按照以下步骤进行操作：
> 1. 首先，进入已克隆的仓库的本地目录。
> 2. 打开终端或命令行界面，并输入以下命令，将远程仓库的 URL 更改为 SSH 形式：
> ```
> git remote set-url origin git@github.com:用户名/仓库名.git
> ```
> 请将 "用户名" 替换为你的 GitHub 用户名，将 "仓库名" 替换为你的仓库名称。
> 3. 现在，你可以使用 SSH 进行推送和拉取操作，而不需要再输入用户名和密码。
> 请注意，如果你之前已经使用 HTTPS 进行了一些操作（如推送或拉取），在切换为 SSH 后，可能需要重新进行身份验证。
